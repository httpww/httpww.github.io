解 $ a x^3 + b x^2 + c x + d = 0 $

$ A = b^2 - 3 a c $

$ B = b c - 9 a d $

$ C = c^2 - 3 b d $

$ \Delta = B^2 - 4 A C $

  1. 当 $ A = B = 0 $

$ x_1 = x_2 = x_3 = \frac{- b}{3 a} = \frac{- c}{b} = \frac{- 3 d}{c} $

  1. 当 $ \Delta > 0 $

$ Y_{ 1 , 2 } = A b + 3 a ( \frac{ - B \pm \sqrt{ B^2 - 4 A C } }{ 2 } ) $

$ x_1 = \frac{ - b - (\sqrt[3]{ Y_1 } + \sqrt[3]{ Y_2 } ) }{ 3 a } $

$ x_{2,3} = \frac{- b - \frac{1}{2} (\sqrt[3]{ Y_1 } + \sqrt[3]{ Y_2 } ) \pm \frac{\sqrt{ 3 } }{2}(\sqrt[3]{ Y_1 } - \sqrt[3]{ Y_2 } ) i}{ 3 a } $

  1. 当 $ \Delta = 0 $

$ K = \frac{B}{A} $

$ x_1 = \frac{- b}{a} + K $

$ x_2 = x_3 = \frac{- K}{2} $

  1. 当 $ \Delta < 0 $

$ T = \frac{2 A b - 3 a B}{2 A \cdot \sqrt{A} } $

$ \theta = arccos \text{ } T $

$ x_1 = \frac{- b - 2 \sqrt{A} cos \text{ } \frac{\theta}{3} }{3 a} $

$ x_{2,3}=\frac{- b + \sqrt{A} (cos \text{ } \frac{\theta}{3} \pm \sqrt{3} sin \text{ } \frac{\theta}{3} )}{3 a} $
